# NetRexx Keywords

Do not use these as variable names, label names, or class/method names.

## NetRexx keywords

```
abstract  adapter   binary    catch      class
depends   do        else      end        exit
extends   finally   if        implements import
indirect  inherits  interface iterate    leave
loop      method    mixin     nop        numeric
options   otherwise package   parse      properties
queue     return    returns   say        select
signals   static    subclass  super      then
this      to        trace     until      uses
volatile  when      while
```

## Special variables

| Variable | Set by |
|----------|--------|
| `rc`     | Host commands |
| `result` | Method return values |

## Notes

- NetRexx is case-sensitive for class and method names
- Java reserved words are also unavailable as identifiers
- `this` refers to the current object (as in Java)
